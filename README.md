# 📺 YouTube Clone Project

## 📌 What this project is
This is a YouTube homepage clone built using HTML, CSS, and JavaScript.  
The goal was to recreate the look and basic functionality of YouTube, including a video grid, sidebar navigation, search system, and shorts section.

It is not connected to the real YouTube API — all videos are manually added using JavaScript objects.

---

## 🎯 What it does

### 🎥 Video Homepage
The main section displays videos in a grid layout similar to YouTube.  
Each video card shows:
- Thumbnail image  
- Video title  
- Channel name  
- Views and upload date  
- Duration  

When clicked, the video opens the real YouTube link.

---

### 🔎 Search Feature
A search bar is included at the top.  
It filters videos in real time based on the title as you type.

---

### 🌙 Dark Mode
A dark mode toggle is included.  
It switches the entire interface between light and dark themes.

---

###  Sidebar Navigation
The sidebar is designed to look like YouTube’s layout.  
It includes:
- Home, Shorts, Subscriptions
- Expandable sections (Subscriptions and You menu)
- Scrollable navigation independent of the main video area

---

### 🎬 Shorts Section
A Shorts section is placed below the main videos.  
It scrolls horizontally and displays short-form video previews.
